# AI Personal Brand Marketing Engine
 
An agentic marketing assistant that analyzes and drafts content. Serves the full-stack container on Hugging Face Spaces!
 
## Tech Stack
* **Frontend:** React, Vite
* **Backend:** Spring Boot, Java 21, Hibernate
* **Database:** H2 Database
 
## Local Development
 
### Frontend
1. Navigate to `/frontend`
2. Run `npm install`
3. Run `npm run dev`
 
### Backend
1. Navigate to `/backend/personal-brand-engine`
2. Run `./mvnw spring-boot:run`
